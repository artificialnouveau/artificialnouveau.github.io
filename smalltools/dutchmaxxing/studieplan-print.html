1. Grammaticaregels

Woordvolgorde

Hoofdzin (V2): persoonsvorm op plaats 2. Ik ga morgen naar huis.
Inversie: begin je niet met het onderwerp, dan werkwoord vóór onderwerp. Morgen ga ik...
Bijzin (omdat, dat, als, terwijl, hoewel, toen, zodat): werkwoord(en) helemaal achteraan. ...omdat ik moe ben.
want vs omdat: want = hoofdzin (gewone volgorde), omdat = bijzin (ww achteraan).
Volgorde bepalingen: Tijd – Manier – Plaats. Ik ga morgen met de trein naar Den Haag.
Vraag: ja/nee-vraag = ww vooraan (Kom je mee?); anders vraagwoord + ww (Waarom kom je?).

Werkwoorden

Perfectum: hebben/zijn + voltooid deelwoord. Beweging/verandering → zijn (ik ben gegaan); de rest → hebben (ik heb gewerkt).
Imperfectum voor achtergrond/gewoonte: het regende, ik woonde daar.
dt-regel: ik → stam; jij/hij/zij/het → stam + t. hij wordt, jij vindt. Deelwoord: 't kofschip → -t.
Scheidbare ww: splitsen in de hoofdzin (ik bel je op), bij elkaar in bijzin/infinitief (...omdat ik je opbel; ik wil je opbellen).
Modaal ww (kunnen, moeten, willen, mogen) + infinitief achteraan: Ik kan morgen komen.
Passief: worden + voltooid deelwoord (de brief wordt verstuurd); verleden werd; voltooid is ... verstuurd.
om te + infinitief = doel; door te + infinitief = middel. om geld te sparen / door te sparen.
dienen te + infinitief = formeel voor moeten: U dient zich te melden.

Naamwoorden & bijvoeglijke naamwoorden

de/het: leer per woord; verkleinwoorden zijn altijd het (het kopje).
Meervoud na telwoord: maat-/eenheidswoorden enkelvoud. drie jaar, twee keer, vijf euro, tien kilo.
Bijvoeglijk nw + e, behalve een het-woord met een: een mooie auto, maar een mooi huis / het mooie huis.

Kleine woorden die punten kosten

Vergelijking: groter dan (niet 'als'); gelijkheid: net zo groot als.
hun / zij / hen: hun = bezit (hun huis); onderwerp = zij/ze; na voorzetsel of lijdend voorwerp = hen.
geen / niet: geen bij zn zonder lidwoord (ik heb geen auto); niet in de rest.
er + voorzetsel: Ik wacht erop. / Waar wacht je op?
Spelling los: te veel, met elkaar, nog steeds, zo'n.
Hoofdletters: talen en landadjectieven: Nederlands, Italiaans. En: 's ochtends, 's avonds.

2. Vaste zinnen & uitdrukkingen

Mening geven & (on)eens zijn

Ik vind dat... / Naar mijn mening... / Ik ben van mening dat...
Ik ben het ermee eens. / Ik ben het er niet mee eens.
Daar ben ik het (niet) mee eens, omdat...
Aan de ene kant... aan de andere kant...
Dat klopt, maar... / Dat is waar, toch denk ik...
Sommigen zeggen X, maar in werkelijkheid...

Aanraden & adviseren

Ik raad je aan om ... te + infinitief.
Het beste is om ... te + infinitief.
Je zou ... kunnen + infinitief.
Ik zou ... (als ik jou was).
Misschien is het handig om ... te + infinitief.
Probeer eens ... / Wat dacht je van ...?

Argumenteren & verbinden (signaalwoorden)

Ten eerste ... ten tweede ... tot slot ...
Bovendien / Daarnaast (toevoegen)
Hoewel ... toch ... / Ondanks dat ... (tegenstelling)
Daarentegen / Echter, (formele tegenstelling)
Namelijk / immers (uitleg)
Daarom / Vandaar dat / Dus (gevolg)
Mits (op voorwaarde dat) / Tenzij (behalve als)
Kortom / Al met al (samenvatten)

Spreken: begin nooit met stilte

Dat is een goede vraag, ... / Nou, ik denk dat ... / Even kijken, ...
Eerlijk gezegd vind ik ... / Wat mij betreft, ... / Wat ik bedoel is ...

Schrijven: aanhef, verzoek & afsluiting

Aanhef formeel: Geachte heer/mevrouw,   informeel: Hallo Anna,
Reden: Ik schrijf u naar aanleiding van ...
Verzoek: Zou u kunnen ...? / Ik zou graag ... / Ik verzoek u vriendelijk om ...
Klacht: Tot mijn spijt moet ik melden dat ... / Helaas ...
Afsluiting formeel: Met vriendelijke groet / Hoogachtend   informeel: Groetjes
Bedanken: Alvast bedankt. Ik hoor graag van u.
