# NT2 B2 woordenschat & structuren (thematisch)

B2 prep. Abstract/formal vocabulary by theme, plus the B2 sentence structures to drill.
Companion to oefen-zwakke-woorden.md (B1 weak spots). Quiz EN->NL.

## Thema: Milieu & duurzaamheid
- climate change -> de klimaatverandering
- emissions -> de uitstoot / de emissies
- sustainable -> duurzaam (sustainability = de duurzaamheid)
- to pollute -> vervuilen ; pollution -> de vervuiling
- the measure -> de maatregel (maatregelen nemen = to take measures)
- renewable energy -> hernieuwbare / duurzame energie
- awareness -> het bewustzijn  (NIET "attentie" = attention)
- waste / garbage -> het afval / het vuilnis
- greenhouse effect -> het broeikaseffect
- to save / economize -> besparen (energie besparen)
- to protect -> beschermen ; het milieu (het-woord!) beschermen

## Algemene B2-woordenschat (gemist in oefeningen)
- to influence -> beïnvloeden (trema op i; invloed = the noun)
- the consequence -> het gevolg / de consequentie
- to emphasize -> benadrukken / nadruk leggen op
- controversial -> omstreden / controversieel
- the policy (government) -> het beleid  (NIET "polis" = insurance policy!)
- inequality -> de ongelijkheid
- to contribute -> bijdragen aan
- the challenge -> de uitdaging

## Anglicismen om te vermijden (kwamen voor)
- "measures" -> maatregelen
- "industry" -> de industrie
- "focuseren" -> focussen op / zich richten op
- "mature" -> volwassen / rijp
- "peerdruk" -> groepsdruk
- "sinds" als "because" -> gebruik omdat/doordat (sinds = alleen tijd)

## B2-zinsstructuren om te drillen
1. Betrekkelijke bijzin (die/dat) = BIJZIN -> werkwoord aan het EIND.
   - de maatregelen die de uitstoot verminderen, zijn duur (niet: die verminderen de uitstoot)
2. Passief = worden + voltooid deelwoord (niet de infinitief).
   - Maatregelen worden (door de overheid) genomen. / verleden: werden genomen.
3. Argumentatie: enerzijds ... anderzijds, met een CONCREET gevolg aan beide kanten.
   - tegenkant met concreet gevolg (bv. "waardoor banen verloren kunnen gaan") scoort op B2.
