# Zwakke woorden om opnieuw te testen (NT2 B1)

Words Ahnjili got wrong or didn't know during vocab drills. Retest these later (EN -> NL).
Most are also added to vocabulary.json so they recur in oefenrondes; this file is the focused retest list.

## Nog zwak (prioriteit)
- to achieve / accomplish -> bereiken / behalen
- to admit -> toegeven
- complicated -> ingewikkeld / complex
- the lack / shortage -> het gebrek / het tekort
- to assume -> aannemen / veronderstellen
- to occur / take place -> plaatsvinden
- to deal with / handle -> omgaan met / aanpakken / behandelen
- to refer to -> verwijzen naar
- to depend on -> afhangen van  (let op: het hangt ervan af)
- the meaning -> DE betekenis (niet "het")

## Look-alike vallen (vaak verward)
- to expect -> verwachten  (NIET verdenken = to suspect)
- to suspect -> verdenken
- society -> maatschappij / samenleving  (NIET gemeenschap = community)
- available -> beschikbaar  (NIET geldig = valid)
- to maintain -> handhaven  (NIET ophouden = to stop)
- nevertheless -> toch / desondanks  (NIET nog = still)
- probably -> waarschijnlijk  (NIET meestal = usually)
- on the other hand -> aan de andere KANT (niet "hand") / daarentegen
- to disappear -> verdwijnen  (NIET verwijden = to widen)

## Eerder gemist, sindsdien goed (af en toe checken)
- beslist = definitely (niet "decision")
- zich aanpassen aan = to adapt to
- to experience -> ervaren / beleven / meemaken
- necessary -> noodzakelijk / nodig
- to encourage -> stimuleren / aanmoedigen (stimuleren met t, niet simuleren)
- to consider -> overwegen
- to require -> vereisen
- to mention -> noemen / vermelden
- to suggest -> suggereren (dubbel -er-)
- to belong to -> behoren tot / horen bij

## Vaste voorzetsels (nog wankel, neigt te vaak naar "op")
- zorgen VOOR (niet op)
- zich interesseren VOOR
- reageren OP (niet naar)
- klagen OVER (niet op)
- lachen OM (niet van)
- bestaan UIT (niet op) = to consist of
- deelnemen AAN
- twijfelen AAN
- vertrouwen OP (rely on) / vertrouwen hebben IN (confidence in)
- zoeken NAAR (search for) / iets OPzoeken (look up)
- verlangen NAAR
- hopen OP
- zich ergeren AAN
- gewend zijn AAN
- bijdragen AAN
- beschikken OVER
- zich voorbereiden OP (niet "voor", ook al lijkt het erop)

### De "aan"-groep (zwakste): denken/deelnemen/twijfelen/ergeren/gewend/bijdragen/lijden/wennen AAN

## Spreken: vaste uitdrukkingen (mening geven/reageren)
- Daar ben ik het (niet) mee eens.  (daar...mee, NIET "dat ben ik het mee eens")
- Ik denk er anders over.  (denken OVER, niet op)
- Naar mijn mening / Volgens mij  (NIET "in mijn mening" = anglicisme)
- Wat mij betreft  = as far as I'm concerned
- Ik zou graag...  = beleefd "I would like" (beleefder dan "ik wil graag")

## Verbindingswoorden (connectoren) - nog zwak
- moreover / in addition -> Bovendien / Daarnaast
- to sum up / in short -> Kortom / Al met al / Samenvattend
- I prefer -> Ik heb liever... / Ik geef de voorkeur aan...
- as a result / consequently -> Daardoor / Als gevolg daarvan
- (nuance) beschrijving in verleden = imperfectum "was/had", gebeurtenis = perfectum "ben/heb gegaan"
- in addition / besides -> Daarnaast / Bovendien / Trouwens
- nevertheless / despite that -> Toch / Desondanks / Niettemin
- namely / that is -> namelijk
- in conclusion / all in all -> Al met al (NIET "al in al" = anglicisme) / Tot slot
- mits = provided that (als en alleen als) ; tenzij = unless (behalve als) -- VAAK VERWARD
- al met al (in short) =/= daardoor (as a result): hou ze los

## Spelling om op te letten
- stijgen, klagen, genoeg, oorzaak, suggereren, gegeten, gewonnen
- gelijk (geen t), de ene kant (niet "enige")
