EmbedClipFarm

Semantic search over YouTube content — search by meaning, not just keywords.

How it works

Step 1
Index videos
CLI pulls transcripts & metadata from YouTube — no video download needed
→
Step 2
Embed
AI embeds transcript chunks into vectors that capture meaning
→
Step 3
Export
Export index as JSON file
→
Step 4
Search here
Load JSON, search by meaning — finds relevant moments even without exact keywords

Quick start

  1. Install: pip install youtube-transcript-api chromadb sentence-transformers
  2. Index a video or playlist: python embedclipfarm.py index "https://youtube.com/watch?v=VIDEO_ID"
    Add --clip for visual search, --whisper for videos without captions optional
  3. Export the index: python embedclipfarm.py export --output index.json
  4. Load it below and start searching!

You can also search directly in the terminal: python embedclipfarm.py search "your query"
